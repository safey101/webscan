# 简单介绍
该工具是一款web扫描工具，功能包括有目录扫描，子域名扫描，c段扫描80端口的web，代理转发，自定义线程等..默认显示标题且对关键词输出红色，比如管理，登录。目前基础功能已经够用，后期会考虑拓展其他功能，例如：爬js敏感信息，目录等。

# 使用效果
这是正在执行目录扫描，默认会输出标题，状态码不同颜色和页面返回长度
```
wscan -u URL地址 -w 字典 
```
